A tool for analyzing cancer data from The Cancer Genome Atlas

Currently, this tool enables visualization of gene expression (RNA-seq) data from primary solid tumor samples via heatmap and violin plot.


Explore the Data:

1) Select cancer type(s)

2) Select clinical feature(s)

3) Select gene(s)

Tip: Begin typing name of gene, and then select gene from dropdown menu.


4) Select mutation(s)
